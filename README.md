# bypy:
 A adfly bypass tool
# Requirements:
 Web Driver (FireFox or Chrome)
 selenium
 urllib3
# Python module installation:
 Windows:
  ```
  pip install -r requirements.txt
  ```
 Linux and Mac:
  ```
  pip3 install -3 requirements.txt
  ```
# Usage:
 Windows:
 ```
 python main.py <adfly-link>
 ```
 Linux and Mac:
 ```
 python3 main.py <adfly-link>
 ```
